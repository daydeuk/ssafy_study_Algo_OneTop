## How To Study
- 스터디 기간 : 2022년 8월 ~ (미정)
- 코드 리뷰 : 수요일 오후 10시
- 언어 : Java  
매일 수업에 대해서 각자 복습, 매주 수 저녁 10시 이후 웹액스를 통해 약 1시간 이상 자신이 맡은 문제에 대해서 집중 리뷰 및 피드백 진행
